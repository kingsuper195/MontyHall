# Monty Hall
A simulator for the famous Monty Hall problem, with two goats and a car. Includes manual choice and automatic.

![](https://raw.githubusercontent.com/kingsuper195/MontyHall/refs/heads/main/GIF-demo.gif)

## [Try It!](https://kingsuper195.github.io/MontyHall/)

## Features
* Automatic and Manual modes.
* Result statictics.
* Configurable.

## How it Works
The project is built using entirely native HTML, Javascript, and CSS, without the use of libraries. The doors are pure CSS with no canvas.
